Question 1:

int sales = 3;		// 3
sales = sales - 5;	// sales = 3 - 5 = -2
sales = sales - 10;	// sales = -2 - 10 = -12
sales = sales + 2;	// sales = -12 + 2 = -10

What is the final value of sales?

Answer: -10

-------------------------------------------------------------------------------------------------------------------

Question 2: 

int apples = 50;	// 50
int goneBad = 5;	// 5
int rotten = 20;	// 20
int sold = 20;		// 20
 	
apples -= goneBad;	// apples = 50 - 5 = 45
apples -= rotten;	// apples = 45 - 20 = 25
apples -= sold;		// apples = 25 - 20 = 5

What is the final value of apples?

Answer: 5

Question 3: 

int score = 0;		// 0
int goal = 1;		// 1
score += goal;		// score = 0 + 1 = 1
score += goal;		// score = 1 + 1 = 2
score += goal;		// score = 2 + 1 = 3
score += goal;		// score = 3 + 1 = 4

What is the final value of score?

Answer: 4

-------------------------------------------------------------------------------------------------------------------

Question 4: 

What's the difference between

goal += 1 and goal = goal + 1 ?

Answer: They are the same. But, the first notation is easier to read.
