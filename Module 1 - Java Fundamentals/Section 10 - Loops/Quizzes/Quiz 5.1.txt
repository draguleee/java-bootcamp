Quiz 5.1

-------------------------------------------------------------------------------------------------------------------

Question 1:

Bart's teacher, Edna, wants him to write this line 99 times.

"I will not forget semi-colons."

Should he use a for loop or a while loop?

Answer: for

Explanation: A for loop runs code a specific number of lines.

-------------------------------------------------------------------------------------------------------------------

Question 2:

The rolling dice function must keep running until the user rolls doubles.

Should one use a for loop, or a while loop?

Answer: while

Explanation: It's impossible to know whe the user will roll doubles. This is a perfect use case for while loops.

-------------------------------------------------------------------------------------------------------------------

Question 3:

You're building a megaphone app, and it should print this message 10 times!

"If Java was easy, they would call it Python!"

Should one use a for loop, or a while loop?

Answer: for

Explanation: The for loop is best for running code a specific number of times.

-------------------------------------------------------------------------------------------------------------------

Question 4: 

You're building an app that sings the song: 99 bottles of beer on the wall.

Should one use a for loop, or a while loop?

Answer: for

Explanation: The for loop will sing the song exactly ninety-nine times.

-------------------------------------------------------------------------------------------------------------------

Question 5:

Your app needs to keep running the "Signing In" code until the user enters their information correctly.

Should one use a for loop, or a while loop?

Answer: while

Explanation: It's unclear when the user will enter the information correctly. So, this is a perfect use case for while loops.

-------------------------------------------------------------------------------------------------------------------

Question 6: 

How many times will the counter i print?

for (int i=0; i<4; i++) {
    System.out.println(i);
}

Answer: four times

-------------------------------------------------------------------------------------------------------------------

Question 7: 

How many times will the counter i print?

for (int i=0; i<=4; i++) {
    System.out.println(i);
}

Answer: five times 

-------------------------------------------------------------------------------------------------------------------

Question 8: 

What is this code's output?

for (int i=0; i<4; i++) {
    System.out.println(i);
}

Answer:
0
1
2
3

Explanation: The loop starts at i=0, and keeps running as long as i is less than 4.
