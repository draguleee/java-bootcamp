Quiz 5.4

-------------------------------------------------------------------------------------------------------------------

Question 1:

while (true) {
   System.out.println("Hi");
}

Answer: runs forever

-------------------------------------------------------------------------------------------------------------------

Question 2:

boolean variable = true;
while (!variable) {		// false
   System.out.println("Hi");
}

Answer: never runs

Explanation: The not operator (!) reverses the boolean to false.

-------------------------------------------------------------------------------------------------------------------

Question 3:

int number = 5;
while (number < 10) {		// true
   System.out.println(number);
   number++;
}

Answer:
5
6
7
8
9

Explanation: The loop keeps running while the number is less than 10.

Question 4: 

Is this a good use case for while loops?

int number = 5;
while (number < 10) {		// true
   System.out.println(number);
   number++;
}

Answer:
Not really. IF you need to run code a specific number of times, it is easier to use a for loop.
