Intermission: print vs println

Before you continue, you need to understand the difference between print() and println().


1. println(): prints text and moves to a new line.

   System.out.println(" a ");
   System.out.println(" b ");
   System.out.println(" c ");

   >> a
   >> b
   >> c


2. print(): prints textbut it does not move to a new line.

   System.out.print(" a ");
   System.out.print(" b ");
   System.out.print(" c ");

   >> a b c
