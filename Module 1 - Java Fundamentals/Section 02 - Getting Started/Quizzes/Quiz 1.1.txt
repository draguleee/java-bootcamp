Question 1: How do you run previous termina commands without re-writing them?
Answer: The up key.

-------------------------------------------------------------------------------------------------------------------

Question 2: File name is Hello.java and the class name is GoodBye. Why does this get an error?
Answer: The file name is not the same as the class name.

-------------------------------------------------------------------------------------------------------------------

Question 3: File name is survey and the class name is survey. There is no error, but do you see a problem?
Answer: The class name should be CamelCase (Survey).

-------------------------------------------------------------------------------------------------------------------

Question 4: What's the output of this code?
	public class Survey {
	    public static void main(String[] args) {
 	        System.out.println("Hi, what's your name?");
		System.out.println("How old are you?");
	    }
	}
Answer:
	>> Hi, what's your name?
	>> How old are you?

-------------------------------------------------------------------------------------------------------------------

Question 5: This code doesn't compile. Why?
	public class Survey {
	    public static void main(String[] args) {
 	        System.out.println("Hi, what's your name?")
		System.out.println("How old are you?")
	    }
	}
Answer: Missing semi-colons.

Question 6: This command: javac <file-name>.java

Question 7: This command: java <file-name>
Answer: runs the compiled code.
